Limit Order Book Simulator Specifications

Introduction

This document outlines the specifications for a limit order book simulator. The simulator will be developed in Python and should support various operations typical to a limit order book, including adding, modifying, and removing orders, as well as processing trades.

Objectives

  • Develop a limit order book simulator in Python.
  • Implement add, modify, and remove order functionalities.
  • Simulate trade matching and order execution.
  • Choose and implement a suitable connection protocol for interaction.

Requirements

The simulator should meet the following requirements:

  • Process and match orders.
  • Provide accurate simulation of a trading environment.
  • Include a user-friendly interface for interaction.
  • Uses TCP/UDP connection protocol.
